# ⚔️ Emoji Wars: Tactical Chat Defense Game

Emoji Wars is a high-density, text-based tactical strategy game engineered to run natively inside a terminal or conversational chat interface. Players manage a dynamic gold economy, construct solid barricades, and deploy specialized forces to defend their Town Hall from expanding waves of cardinal-pathing invaders.

## 🕹️ System Features & Mechanics

* **Condensed UI Layout**: Fully optimized 10x8 grid canvas paired with a zero-scroll side dashboard tracking real-time status telemetry.
* **Delta Exception Health Radar**: The system sidebar stays completely clear during optimal conditions, dynamically flashing damage alerts only when forces sustain active wounds.
* **Cardinal Physics Engine**: Invaders move strictly along cardinal coordinates lines (Up, Down, Left, Right). Cell boundaries enforce absolute entity separation, mechanically shoving colliding entities to side channels.
* **Consolidated Taskbar Shop**: A dynamic inventory tier list that automatically screens out high-cost items, rendering only options matching your live liquid gold wallet balance.
